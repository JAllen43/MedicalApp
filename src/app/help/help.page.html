Help

Please find some pointers on how to use the app below:

Information Button

Pressing this button will display a description of the drug that it is listed under

 Adding Medicine to your Tracker

To add the relevant medicine to your Tracker, this toggle switch will need to be tapped upon so that it turns blue
Items that appear in your tracked list can be used to set a timer and reminders of when medication is required to be taken

Remove Medicine

This allows you to remove the medicine from the relevant screen, whether it be deleting an entry after finishing a course of medication or removing it from the application completely
